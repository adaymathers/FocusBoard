Configuración para GitHub Pages

Pasos para publicar en GitHub Pages:

  1. Crear repositorio en GitHub:
    • Nombre: FocusBoard o focusboard-landing
    • Hacer público el repositorio
  2. Subir archivos:
    • index.html (página principal)
    • Carpeta gallery/ con capturas
    • favicon.png (opcional)
  3. Habilitar GitHub Pages:
    • Ir a Settings del repositorio
    • Scroll down a "Pages"
    • Source: Deploy from a branch
    • Branch: main / root
    • Save
  4. Personalizar URLs:
    • Actualizar enlaces de descarga en index.html
    • Cambiar "tu-usuario" por "adaymathers"
    • URL final: https://adaymathers.github.io/FocusBoard

Archivos a personalizar:

En index.html:

  • Línea con href="https://github.com/adaymathers/FocusBoard/releases/latest/download/FocusBoard-1.0.0-setup.exe"
  • Ya actualizado con "adaymathers"
  • Meta tag og:url actualizado
  • Enlaces de GitHub en footer actualizados

Opcional - Dominio personalizado:

Si tienes un dominio propio, crea un archivo CNAME con tu dominio.
